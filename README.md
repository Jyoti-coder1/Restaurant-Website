# HealthyHub – Responsive Restaurant Website

HealthyHub is a modern, fully responsive restaurant website built using HTML, CSS, and JavaScript.  
The project focuses on clean UI design, smooth user experience, and essential front-end interactivity suitable for real-world restaurant or food business websites.

---
## Live Preview
https://restaurant-website-hazel-zeta.vercel.app/

---

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Mobile-friendly hamburger menu
- Interactive food category section
- Clean and structured food menu layout
- Customer testimonials section
- Contact form with JavaScript validation
- Accessible and SEO-friendly HTML structure

---

## Tech Stack

- HTML5 – Semantic and accessible markup  
- CSS3 – Flexbox, media queries, animations  
- JavaScript – DOM manipulation and user interaction  

---

## How to Use

1. Open the live website link provided above.
2. Navigate using the menu bar (Home, About, Menu, Contact).
3. Scroll through different sections to explore food categories and menu items.
4. On mobile, click the hamburger menu to open navigation.
5. Fill the contact form and submit your message (client-side validation enabled).

---

## JavaScript Functionality

- Contact form validation (name, email, message)
- Prevents empty or invalid form submissions
- Improved UX through client-side interactivity
- Enhances usability without any external libraries
